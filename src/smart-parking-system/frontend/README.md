# Smart Parking System Frontend

This is the frontend part of the Smart Parking System project. It is built using HTML, CSS, and JavaScript, and interacts with the backend developed in Java Spring Boot.

## Project Structure

- `src/`
  - `index.html`: The main HTML file that serves as the entry point for the web application.
  - `styles.css`: Contains the styles for the web application.
  - `app.js`: JavaScript file that handles the logic for interacting with the backend API.

## Features

- Add parking slots
- Park vehicles
- Remove vehicles

## Getting Started

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the frontend directory:
   ```
   cd smart-parking-system/frontend
   ```

3. Open `index.html` in a web browser to view the application.

## Dependencies

This project does not have any external dependencies for the frontend. All functionalities are implemented using vanilla JavaScript, HTML, and CSS.
